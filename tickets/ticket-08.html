Основные определения и факты

Матрица Грама системы $x_1,\ldots,x_k$: $G=((x_i,x_j))$.
Если $X$ — матрица столбцов координат, то $G=X^TX$, поэтому $\det G\ge0$.
Грам–Шмидт вычитает из очередного вектора его проекцию на уже построенное подпространство.
Для полной системы $\sqrt{\det G}$ — объём параллелепипеда.

Примеры

  1. Для ортонормированной системы матрица Грама равна $I$.
  2. Для $x_1=(1,0)$, $x_2=(1,1)$: $G=\begin{pmatrix}1&1\\1&2\end{pmatrix}$, $\det G=1$.
  3. Грам–Шмидт для $(1,0),(1,1)$ даёт ортогональные $(1,0),(0,1)$.
