# Алгебра II — GitHub Pages

Готовый статический сайт с конспектом по билетам 1–27.

## Публикация

Вариант 1: загрузить всё содержимое архива в корень репозитория и включить GitHub Pages из ветки `main`, папка `/root`.

Вариант 2: оставить workflow `.github/workflows/pages.yml` и в настройках Pages выбрать **GitHub Actions**.

## Что внутри

- `index.html` — главная и поиск.
- `tickets/` — страницы 27 билетов.
- `map.html` — карта тем.
- `trainer.html` — карточки для самопроверки.
- `interactives/` — интерактивные визуализации.
- `assets/lin_alg_notes.pdf` и `assets/lin_alg_notes.tex` — исходный конспект.
